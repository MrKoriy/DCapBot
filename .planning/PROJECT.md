# DCapBot — Telegram Subscription Bot

## What This Is

Telegram-бот на aiogram для управления платным доступом в закрытый Telegram-канал. Бот принимает оплату через ЮKassa, автоматически выдаёт инвайт-ссылку, следит за сроком подписки, напоминает о продлении и кикает неоплативших участников. Админ-панель встроена в бота.

## Core Value

Пользователь оплачивает подписку и мгновенно получает доступ в канал — без ручной обработки заявок владельцем.

### Out of Scope

- Веб-интерфейс админ-панели — избыточно для одного канала, достаточно бота
- OAuth/внешняя авторизация — работаем через Telegram ID
- Поддержка нескольких каналов — бот для одного канала
- Возврат платежей через бота — обрабатывается вручную через ЮKassa

## Context

- Заказчик ведёт платный Telegram-канал и хочет автоматизировать приём оплаты и управление доступом
- Оплата исключительно через ЮKassa (российская платёжная система)
- Сроки подписки: 1 и 3 месяца, цены задаются через конфигурацию
- Бот будет единственной точкой входа для пользователей — нет сайта или лендинга
- Хостинг на VPS (Ubuntu), бот как systemd-сервис

## Constraints

- **Tech stack**: Python 3.11+, aiogram 3.x, PostgreSQL, asyncpg/SQLAlchemy
- **Платёжная система**: ЮKassa (yookassa-sdk-python)
- **Деплой**: VPS, systemd-сервис
- **БД**: PostgreSQL — выбор заказчика
